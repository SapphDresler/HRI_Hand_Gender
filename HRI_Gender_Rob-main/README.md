# HRI_Gender_Rob
---
## Test Scripts

### test_haar_face.py
**Purpose:**  
Tests real-time face detection using a camera and a Haar Cascade model.  
This script verifies that the camera works correctly and that human faces can be detected and displayed.

### test_face_gender.py
**Purpose:**  
Tests the full pipeline of face detection and gender classification.  
After detecting a face, the script classifies it as **male** or **female** using a TensorFlow Lite model.
